# 🎮 Татар Уены — Игровой Telegram бот

Татарская игра для компании с двумя режимами.

## Игры
- 🧠 **Викторина** — 20 вопросов про татарскую культуру, кто первый ответит — получает очко
- 😄 **Я никогда не...** — 20 татарских утверждений, нажимай если делал

## Запуск
1. Получи токен у @BotFather
2. Добавь в Railway переменную `BOT_TOKEN`
3. Задеплой на Railway

## Как использовать
Добавь бота в групповой чат → /start → выбери игру → зови друзей!
